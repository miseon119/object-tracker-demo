# Object-Tracker-demo

Multiple object trackers using opencv 4.5.1

## Preinstallation

opencv-python 4.5.1
```console
$ pip install opencv-python
``` 

opencv contrib python
```console
$ pip install opencv-contrib-python
```

## 8 Types Object Trackers 

- BOOSTING
- MTL
- KCF
- TLD
- MEDIANFLOW
- GOTURN
- MOSSE
- CSRT
